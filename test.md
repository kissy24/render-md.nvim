# Notion-like Rendering in Neovim

Welcome to **render-md.nvim**.

## Features
- Real-time rendering
- Treesitter based parsing
- Extmarks and Virtual Text

### Getting Started
1. Install the plugin
2. Open a markdown file
3. Enjoy the view!

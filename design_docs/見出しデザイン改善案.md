# 見出しデザイン改善案：render-md.nvim

## 1. 目的
現在の見出し装飾は背景色とアイコンのみですが、より Notion の持つ「ゆとり」と「構造の明確さ」を再現するために、高度な装飾を追加します。

## 2. デザインの工夫
### H1: ページタイトル風
- **アイコン**: より大きな存在感を持たせる。
- **下線 (Underline)**: 見出し行のすぐ下の行に、仮想テキスト（Extmarks）を使用してボーダーラインを表示。
- **スペーシング**: 見出しの上に1行分の仮想的な余白を追加し、文書の区切りを強調。

### H2: セクション区切り
- **左ボーダー**: 行の左端に強調カラーの太い垂直線を配置。
- **背景色**: H1よりは控えめだが、セクションとして認識しやすい色。

### H3: 小見出し
- **インデント**: H1/H2よりも少し右に寄せることで、階層構造を視覚的に表現。
- **フォントスタイル**: 背景色は使わず、色と太字のみで表現。

## 3. 技術的アプローチ
- `nvim_buf_set_extmark` の `virt_lines` を使用して、行間に境界線や余白を挿入する。
- `virt_text_win_col` を使用して、ウィンドウの幅いっぱいに広がる背景色や下線をシミュレートする。

## 4. 実装計画
1.  **ハイライト定義の更新**: よりモダンなパステルカラー/ダークモード向けの配色に変更。
2.  **H1の下線実装**: `virt_lines` を用いた境界線の表示。
3.  **階層別インデント**: 見出しレベルに応じた `virt_text` のスペース調整。

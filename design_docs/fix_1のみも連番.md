# 数字の箇条書きの連番対応

## 概要

Markdownの数字の箇条書きで、以下のように同じ数字が続く場合に、連番として解釈しレンダリングする。

### 元のMarkdown

```markdown
1. a
1. b
1. c
```

### 期待される表示

1. a
2. b
3. c

## 仕様

- `render-md`のレンダリング処理において、数字の箇条書き (`1.`, `2.`など) を検出する。
- 箇条書きのリストが続く限り、カウンターをインクリメントし、その値を表示に使用する。
- リストが途切れたら、カウンターをリセットする。
- 箇条書きの数字が`1.`以外から始まる場合も考慮する。

例:
```markdown
3. a
3. b
3. c
```

期待される表示:

3. a
4. b
5. c
